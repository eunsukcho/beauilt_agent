# 데이터 폴더

이 폴더에는 벡터 DB에 넣을 데이터 파일들이 저장됩니다.

## 폴더 구조

```
data/
├── DATA_GUIDE.md           # 데이터 관리 종합 가이드 (저작권, 자료 출처, 확장 가이드)
├── 피부관리의기초.md        # 핵심 피부 관리 지식 베이스
├── skin_type_cards/        # 피부 타입/상태 진단 카드
│   ├── 01_건성피부.md
│   ├── 02_지성피부.md
│   ├── 03_복합성피부.md
│   ├── ...
│   └── REFERENCES.md
└── ingredient_cards/        # 성분 정보 카드
    ├── 01_히알루론산.md
    ├── 02_세라마이드.md
    └── 03_나이아신아마이드.md
```

## 주요 파일 설명

### DATA_GUIDE.md
데이터 추가 및 관리에 필요한 모든 가이드를 포함합니다:
- 저작권 및 데이터 사용 가이드
- 공인된 자료 목록 (공공누리, 학회 자료 등)
- 데이터 확장 가이드
- 성분 데이터 전략 (벡터 DB vs API)

### 피부관리의기초.md
대한피부과학회 자료를 참고하여 작성한 핵심 피부 관리 지식 베이스입니다.

### skin_type_cards/
피부 타입과 상태를 진단하기 위한 카드들입니다.

### ingredient_cards/
주요 성분에 대한 상세 정보를 담은 카드들입니다.

## 새로운 데이터 추가하기

1. 적절한 폴더에 새 마크다운 파일 생성
2. `DATA_GUIDE.md`의 가이드를 참고하여 작성
3. `uv run python scripts/init_vector_db.py` 실행하여 벡터 DB에 추가

## 벡터 DB 업데이트

데이터를 추가하거나 수정한 후:

```bash
uv run python scripts/init_vector_db.py
```

이 명령어는 모든 카드와 문서를 벡터 DB에 추가합니다.
