# 챗봇 사용 예시 질문 💬

챗봇에 물어볼 수 있는 질문 예시입니다. 자연스럽게 대화하듯 질문하시면 됩니다!

## 🩺 피부 상태 진단 관련

### 기본 진단 질문
- "제 피부가 자꾸 번들거려요"
- "세안 후에 피부가 당겨요"
- "트러블이 자주 생겨요"
- "피부가 건조한 것 같아요"
- "제 피부 타입이 뭔지 모르겠어요"
- "T존은 기름지고 볼은 건조해요"
- "피부가 자주 붉어지고 따가워요"
- "각질이 많이 일어나요"

### 구체적인 증상 설명
- "사무실에 있으면 피부가 너무 건조해져요. 기름은 안 나오는데 당김이 느껴져요"
- "아침에 일어나면 이마와 코가 번들번들한데, 볼은 건조해요"
- "세안 후에 피부가 당기고, 메이크업이 잘 들뜨는 편이에요"
- "트러블이 생기면 자주 반복되고, 피지가 많은 것 같아요"

## 📊 종합 분석 관련

### 생활 패턴 고려 질문
- "야근을 자주 하는데 피부 관리 어떻게 해야 할까요?"
- "운동을 많이 하는데 피부가 건조해져요"
- "스트레스를 많이 받으면 피부가 더 안 좋아져요"
- "수면 시간이 부족한데 피부 관리 팁 알려주세요"

## 📝 케어 가이드라인 관련

### 루틴 질문
- "건조한 피부 케어 방법 알려주세요"
- "지성 피부 아침 루틴 추천해주세요"
- "민감성 피부 케어 가이드 알려주세요"
- "복합성 피부 관리 방법이 궁금해요"
- "트러블 피부 케어 루틴 추천해주세요"

### 구체적인 상황 질문
- "사무실에서 건조해질 때 어떻게 하면 좋을까요?"
- "운동 후 피부 관리 어떻게 해야 해요?"
- "여행 갈 때 피부 관리 팁 알려주세요"
- "겨울철 건조한 피부 케어 방법이 궁금해요"

### 제품 사용법 질문
- "세안 후 바로 보습제를 발라야 하나요?"
- "토너와 세럼 순서가 어떻게 되나요?"
- "미스트를 자주 뿌려도 되나요?"
- "각질 제거는 얼마나 자주 해야 하나요?"

## 🧪 성분 분석 관련

### 성분 효능 질문
- "히알루론산 효능이 뭐예요?"
- "세라마이드가 건조한 피부에 좋나요?"
- "나이아신아마이드 추천해주세요"
- "제 피부에 맞는 성분 알려주세요"

### 성분 조합 질문
- "히알루론산과 세라마이드를 함께 써도 되나요?"
- "나이아신아마이드와 레티놀 함께 사용 가능한가요?"
- "어떤 성분들을 함께 쓰면 좋을까요?"
- "함께 쓰면 안 되는 성분 조합이 있나요?"

### 제품 선택 질문
- "건조한 피부에 좋은 성분이 뭐예요?"
- "지성 피부에 피해야 할 성분이 있나요?"
- "민감성 피부에 안전한 성분 추천해주세요"

## 🔮 시뮬레이션 관련

### 변화 예측 질문
- "이 루틴 적용하면 어떻게 될까요?"
- "케어 가이드대로 하면 언제쯤 개선될까요?"
- "변화 예측해주세요"
- "1개월 후 피부 상태가 어떻게 될까요?"
- "시뮬레이션 해주세요"

## 💬 추가 질문 (이전 대화 맥락 활용)

### 맥락 기반 질문
- "그럼 미스트는 언제 뿌리면 좋을까요?"
- "추가로 궁금한 게 있는데요"
- "그 성분은 어떻게 사용하나요?"
- "그 방법이 제 피부에 맞을까요?"
- "더 자세히 알려주세요"

## 🎯 실전 사용 예시

### 예시 대화 1: 피부 진단 → 케어 가이드
```
사용자: "제 피부가 자꾸 번들거리고 트러블이 자주 생겨요"
→ 챗봇: 피부 진단 → 분석 → 케어 가이드 제공

사용자: "그럼 아침 루틴은 어떻게 하면 좋을까요?"
→ 챗봇: 이전 맥락을 참고하여 구체적인 아침 루틴 안내
```

### 예시 대화 2: 성분 질문
```
사용자: "히알루론산 효능이 뭐예요?"
→ 챗봇: 히알루론산 성분 분석 제공

사용자: "세라마이드와 함께 써도 되나요?"
→ 챗봇: 성분 조합 가이드 제공
```

### 예시 대화 3: 상황별 질문
```
사용자: "사무실에 있으면 피부가 너무 건조해져요"
→ 챗봇: 진단 → 분석 → 사무실 환경에 맞는 케어 가이드

사용자: "미스트를 뿌려도 되나요?"
→ 챗봇: 이전 맥락 참고하여 미스트 사용법 안내
```

## 💡 질문 팁

1. **자연스럽게 질문하세요**: "제 피부가..."처럼 자연스러운 표현으로 질문하면 더 정확한 답변을 받을 수 있습니다.

2. **구체적으로 설명하세요**: 증상이나 상황을 구체적으로 설명하면 더 맞춤형 조언을 받을 수 있습니다.
   - 좋은 예: "사무실에 있으면 피부가 건조해지는데, 기름은 안 나와요"
   - 나쁜 예: "피부가 안 좋아요"

3. **추가 질문을 활용하세요**: 이전 대화 맥락을 기억하므로 "그럼", "추가로" 같은 표현으로 추가 질문이 가능합니다.

4. **기능별로 질문하세요**:
   - 진단: "제 피부 타입이 뭐예요?"
   - 가이드: "케어 방법 알려주세요"
   - 성분: "히알루론산이 뭐예요?"
   - 시뮬레이션: "변화 예측해주세요"

---

**참고**: 챗봇은 이전 대화 맥락을 기억하므로, 진단 후 추가 질문을 하면 더 구체적인 답변을 받을 수 있습니다!
